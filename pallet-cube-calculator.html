How we compute cube

Pallet cube (volume) is length × width × estimated total height. Total height equals pallet base height (if provided) plus either your max stack height or layers × box height when layers are used and a box height is given. If no box height is provided with "layers" mode, height is estimated from layers × an entered average layer height (not shown here for simplicity).

FAQ

Why is utilization blank?

We need your box dimensions to compute TI (per‑layer boxes), estimated layers, and total box volume relative to pallet cube.

Standards included

Presets include EU (1200×800, 1200×1000), US (48×40 / 1219×1016), AU (1165×1165), JP (1100×1100). You can input any custom size.
